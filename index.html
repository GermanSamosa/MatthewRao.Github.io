MoonRise Movies

C# | Razor Pages | ASP.NET Core | EntityFramework | Google Charts API | Stripe | Microsoft SQL Server | Azure Storage Blobs | Azure Application | Microsoft Azure Storage Explorer

Purpose of the application is to let movie lovers skip the line by register/loging in and booking tickets and reviewing their favourite movies.

  • Team project, my specialty was UI and front-end data manipulation
  • User can register and login, view movie trailers, reviews and change their password. They will also recieve an email once tickets are booked.
  • Admin can create, read, update and delete movies, movie categories and users. They can also view and manager charts based on the industry.
  • Admin can also upload pictures for upcoming movies they want to add.
  • Analytic Section, Charts and Statistics using google charts api
